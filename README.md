# Farm-ish

2D farm-style game in Unity.

## Tech Stack

- **Unity 6** (6000.3.8f1) – Game engine
- **2D Sprite** – Sprite rendering
- **2D Tilemap** – Grid-based level layout (grass, dirt, etc.)
- **2D Tilemap Extras** – Extra tile types and brushes
- **2D Animation** – Sprite animation
- **2D Aseprite** – Import .ase/.aseprite files from Aseprite
- **2D SpriteShape** – Vector-based shapes for terrain/rivers
- **2D PSD Importer** – Import layered Photoshop files
- **Input System** – New input API (keyboard, mouse, gamepad)
- **Universal Render Pipeline** (URP) – Modern rendering pipeline
- **Cinemachine** – Camera control and transitions
- **Timeline** – Cutscenes and sequenced events
- **UGUI** – Unity’s UI system (buttons, panels, etc.)
- **Visual Scripting** – Node-based scripting (optional)
